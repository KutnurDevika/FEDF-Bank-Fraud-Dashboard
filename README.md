# 🛡️ Bank Fraud Detection Alert Dashboard

A modern React-based fraud monitoring dashboard that simulates banking security operations through authentication, fraud alerts, analytics, reports, API integration, and real-time monitoring workflows.

## 🌐 Live Demo

[Live Application](https://fedf-bank-fraud-dashboard.vercel.app/)

## 🚀 Live Features

### Authentication System
- User Signup
- User Login
- Protected Dashboard Access
- Protected Reports Access
- Logout Functionality
- Session Persistence using Local Storage

### Fraud Monitoring Dashboard
- Real-Time Fraud Alert Management
- Fraud Risk Analysis
- Security Monitoring Section
- Banking Security Insights
- Recent Activity Tracking
- Recent Transactions Monitoring
- Fraud Analytics Visualization using Recharts
- Monthly Fraud Trend Analysis Chart

### Alert Management
- Generate Fraud Alerts
- Search Alerts by Alert ID
- Filter Alerts by Risk Level
- Persistent Alert Storage using Local Storage
- Export Alert Data as JSON

### Reports System
- Fraud Statistics Overview
- Alert Summary Reports
- Dashboard Navigation to Reports

### User Experience Features
- Password Visibility Toggle
- Form Validation
- Loading States
- Dynamic Navigation
- Responsive Dashboard Layout

### API Integration
- Fetch API Integration
- Dynamic Analyst Data Rendering
- Real-Time Data Handling using useEffect

### Cloud & DevOps
- Deployed on Vercel
- CI/CD Pipeline (Future Enhancement)
- Automated Testing
- Performance Monitoring
- Production Logging

### React Features Used
- React Components
- React Router DOM
- Protected Routes
- useState Hook
- useEffect Hook
- Conditional Rendering
- Event Handling
- Dynamic Rendering using map()
- API Integration using Fetch API
- Local Storage Persistence
- Recharts Data Visualization
